Приложение для расчета параметров оптимального обнаружителя на основе коэффициента эксцесса и коэффициента коррекляции. Дополнительная информация в [документации](docs/Оптимальный обнаружитель.docx)

Для сборки:
```shell
yum install cmake
cd build
cmake ..
make
```
